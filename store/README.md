# Store Directory

This directory contains uploaded Minecraft Education world files.

## Purpose
- Stores uploaded .mcworld and .mctemplate files
- Files are automatically organized by timestamp
- Contains the working storage for all uploaded worlds

## Contents
- `.mcworld` files: Minecraft Education world files
- `.mctemplate` files: Minecraft Education template files

## Note
**This directory is ignored by Git** to prevent large files and user data from being committed to the repository.

## File Structure
Files are stored with timestamp prefixes:
```
YYYYMMDD_HHMMSS_original_filename.mcworld
```

Example:
```
20231029_143022_MyWorld.mcworld
```
